is-regexp

Check whether a variable is a regular expression

Install

$ npm install --save is-regexp

Usage

var isRegexp = require('is-regexp');
isRegexp('unicorn');
//=> false
isRegexp(/unicorn/);
//=> true
isRegexp(new RegExp('unicorn'));
//=> true
